# Bootstrap Internship UI Project

A small static site built using Bootstrap 5. Demonstrates remixing Bootstrap components into original, responsive pages.

## Pages
- `index.html` — Home (navbar, hero, cards, carousel)
- `about.html` — About / services
- `contact.html` — Contact page with working form (Netlify or Formspree)

## Tech
- HTML5, CSS3
- Bootstrap 5 (via CDN)
- Git + GitHub
- Optional deployment: GitHub Pages / Netlify

# from project root
python3 -m http.server 8000
